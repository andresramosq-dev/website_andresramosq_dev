Cada nota es un archivo en `src/content/blog/`.

## Pasos

1. Crea un `.md` nuevo.
2. Completa título, descripción, fecha y categoría.
3. Escribe en Markdown.
4. Corre `npm run dev` o `npm run build`.

Astro genera la página automáticamente.
