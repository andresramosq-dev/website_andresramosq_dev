Cada nota es un archivo en `src/content/blog/`.

## Pasos

1. Crea un `.md` nuevo.
2. Completa título, descripción y fecha.
3. Escribe el contenido en Markdown.
4. Corre `npm run dev` o `npm run build`.

Astro genera la página en `/blog/nombre-del-archivo`.
